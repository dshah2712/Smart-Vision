# An Application for Visually impaired people for visual Assitance

- tensorflow
- cv2
- django
- pytorch
- os
- pytesseract
- pyttsx3
- numpy 
- tarfile
- zipfile
- time
- engineio
- matplotlib
- Protobuf 3.0.0
- Python-tk
- Pillow 1.0
- lxml
- tf-slim
- slim
- Cython
- contextlib2
- cocoapi
